Project-Voice
=============

The voice project of Embedded System use STM32

* wiki website : https://github.com/DewerZHT/Project-Voice.wiki.git

Hardware
========

* STM32F407VG Core Board
* STM32F4 Discovery
* QCopter MachineVision

Software Tools
==============
* Program IDE
  * Code::Blocks
  * Keil C
* Hardware design
  * Altium Designer
  * LTSPICE
* Documents
